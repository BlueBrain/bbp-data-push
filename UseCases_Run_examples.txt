## Execution examples

Then bba-data-push is now added to your PATH and can be executed from within a new terminal window. Some execution examples :


### Push into Nexus with their provenance, two atlas pipeline volumetric datasets with a resolution of 25 um:
A 'cell_density' folder containing volumetric cell densities files.
An hybrid annotation volume file from ccfv2 and ccfv3.

bba-data-push --forge-config-file ../forge-tests-config.yaml \
              --nexus-env staging \
              --nexus-org bbp \
              --nexus-proj atlas \
              --nexus-token-file atlas \
push-volumetric --dataset_path '../cell_density' \
                --dataset_path '../annotation_hybrid.nrrd' \
                --config '../push_dataset_config.yaml' \
                --provenance "atlas-building-tools combination combine-annotations:atlas-building-tools, version 0.0.1," \
                --provenance "atlas-building-tools cell-densities glia-cell-densities:atlas-building-tools, version 0.0.1" \
                --voxels_resolution '25'



### Push into Nexus a folder filled with obj meshes based in the parcellation volume resulting of the hybridation between CCFv2 and CCFv3 with the isocortex layer 2 and 3 split:

bba-data-push --forge-config-file ../forge-tests-config.yaml \
              --nexus-env staging \
              --nexus-org bbp \
              --nexus-proj test_push \
              --nexus-token-file atlas \
push-meshes --dataset_path '../brain_region_meshes_l23split' \
                --config '../push_dataset_config.yaml' \
                --hierarchy_path '../hierarchy_l23split.json.json' \


### Push into Nexus with its provenance, an atlas sonata h5 file storing the 3D cell positions and orientations at a resolution of 25 um:

bba-data-push --forge-config-file ../forge-tests-config.yaml \
              --nexus-env staging \
              --nexus-org bbp \
              --nexus-proj atlas \
              --nexus-token-file atlas \
push-volumetric --dataset_path '../cell_records_sonata.h5' \
                --config '../push_dataset_config.yaml' \
                --hierarchy_path '../hierarchy.json' \
                --provenance "brainbuilder cells positions-and-orientations:brainbuilder cells positions-and-orientations, version 2," \
                --voxels_resolution '25'

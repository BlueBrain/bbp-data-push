###### Concatenate the Base command with one of these:

### Push a hybrid annotation file and a density folder:

  push-volumetric \
  --dataset_path '/cell_density' \
  --dataset_path '/annotation_hybrid.nrrd' \
  --hierarchy_path 'hierarchy.json' \
  --config '/push_dataset_config_template.yaml' \
  --provenance "atlas-building-tools combination combine-annotations:atlas-building-tools, version 0.0.0.dev6," 
  --voxels_resolution '25'

 push-meshes \

###### Concatenate the Base command with one of these:

### Push a ccfv2 annotation file with a contributor:

  push-volumetric \
  --nrrd_local_path "../tests/data/annotation_25.nrrd" \
  -desc "Original Allen ccfv2 annotation volume at 25 microns" \
  -extratypes "BrainParcellationDataLayer" \
  -voxeltype label \
  -idatlasrelease "https://bbp.epfl.ch/neurosciencegraph/data/dd114f81-ba1f-47b1-8900-e497597f06ac" \
  --id_spatial_ref "https://bbp.epfl.ch/neurosciencegraph/data/allen_ccfv3_spatial_reference_system" \
  --contributor_name "Jean-Vincent"


### Push a astrocytes density volume and add a version tag:

  --nrrd_local_path "../tests/data/astro_density.nrrd" \
  -desc "Cell density volume for astrocytes at a resolution of 25 microns. Output of the cell density pipeline by Csaba Eroe" \
  -extratypes "GliaCellDensity" \
  -extratypes "CellDensityDataLayer" \
  -voxeltype intensity \
  -idatlasrelease "https://bbp.epfl.ch/neurosciencegraph/data/e2e500ec-fe7e-4888-88b9-b72425315dda" \
  --id_spatial_ref "https://bbp.epfl.ch/neurosciencegraph/data/allen_ccfv3_spatial_reference_system" \
  --version 'myversion'



### Push a batch of 3 meshes with 2 extra informations:

 push-meshes \
